# Free Claude Cowork Course

**Learn Claude Cowork by Doing** - The only interactive course that teaches you Claude Cowork inside Claude Cowork itself.

## What is This?

This is a free, hands-on course that teaches you how to use Claude Cowork (the desktop app) for real work tasks. No coding required.

You'll learn by doing - not watching videos. Claude itself teaches you interactively as you work through realistic scenarios.

## What You'll Learn

### Module 1: Working with Claude (2-3 hours)
Learn the core skills by working through your chosen scenario:

- **Navigate files** - Explore and understand any folder structure
- **Extract insights** - Pull structured info from messy documents
- **Synthesize patterns** - Find themes across multiple files
- **Analyze images** - Get insights from screenshots and designs
- **Research the web** - Search and fetch content from websites
- **Use agents** - Process many tasks in parallel
- **Create documents** - Generate Word, Excel, and PowerPoint files
- **Get feedback** - Use multiple perspectives to review your work

### Module 2: Extending Claude (1-2 hours)
Connect Claude to your real tools and build workflows for your actual job:

- **Connectors** - Link Claude to Slack, Notion, Google Calendar, and more
- **Plugins** - Install domain-specific toolkits for your role
- **Skills & Commands** - Use slash commands for role-specific workflows
- **Customization** - Tailor Claude's output to your voice and preferences
- **Real Workflows** - Build a working workflow you'll use on Monday morning

## Getting Started

1. **Download** the course files from [freecoworkcourse.com](https://freecoworkcourse.com)
2. **Unzip** by double-clicking the downloaded file
3. **Open** the Claude desktop app ([download here](https://claude.ai/download))
4. **Go to** the Cowork tab (look for the Cowork icon in the left sidebar)
5. **Say** "I downloaded a Claude Cowork course. Can you start it?"
6. **Select** the unzipped course folder when prompted

Claude takes it from there - it'll read the course files and jump right into Lesson 1.1.
