## When to Use Each Tool

**Web Search:**
- General research questions
- Best practices and trends
- When you need multiple perspectives
- Current events and recent information

**Page Fetching:**
- Specific websites you want to analyze
- Competitor research (text content)
- Extracting data from a known source
- When you need detailed content from one page

**Browser Screenshots:**
- Visual competitive analysis
- Understanding design and UX choices
- When you need to SEE what competitors are doing
- Capturing specific pages for reference

**The Power Combo:**
Search → Fetch → Screenshot → Document
